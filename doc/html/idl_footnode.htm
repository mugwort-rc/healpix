... party1
An extensive list of third party IDL libraries can be found at https://idlastro.gsfc.nasa.gov/other_url.html. Among them, we recommend the IDL Astronomy Users library (https://idlastro.gsfc.nasa.gov) maintained by Wayne Landsman and which is extensively used in HEALPix-IDL, David Fanning's IDL Coyote library (http://www.idlcoyote.com) from which some routines are used in HEALPix-IDL, and Craig Markwardt's IDL library (https://cow.physics.wisc.edu/~craigm/idl/idl.html).
