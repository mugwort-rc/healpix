ang2vec

Routine to convert the position angles of a point on the sphere into its 3D position vector (x,y,z) with , , .


Location in HEALPix directory tree: src/f90/mod/pix_tools.f90

FORMAT

call ang2vec( theta, phi, vector )

ARGUMENTS

name & dimensionality kind in/out description
       
theta DP IN colatitude in radians measured southward from north pole (in ).
phi DP IN longitude in radians measured eastward (in ).
vector(3) DP OUT three dimensional cartesian position vector (x,y,z) normalised to unity. The north pole is (0,0,1)
